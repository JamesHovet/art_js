# p5.axidraw API Reference

Source: https://jmpinit.github.io/p5.axidraw  
Version: 1.0.0 (prerelease-1)

## Setup (this repo)

Load as a `<script>` tag — do NOT import through webpack (see memory for why).

```html
<script src="/vendor/p5.axidraw.js"></script>
```

```typescript
declare const axidraw: { AxiDraw: new () => any };
const axi = new axidraw.AxiDraw();
```

## Connection

| Method | Returns | Notes |
|--------|---------|-------|
| `connect()` | `Promise<void>` | Pops browser serial port dialog. Must be called from a user gesture (click). Chrome/Edge only. |
| `disconnect()` | `Promise<void>` | Close the serial connection. |

## Motor control

| Method | Returns | Notes |
|--------|---------|-------|
| `enable()` | `Promise<void>` | Lock motors (movement commands do this automatically). |
| `disable()` | `Promise<void>` | Release motors so head can be moved by hand. Call when done. |
| `isBusy()` | `boolean` | True if commands are still executing. |

## Drawing

| Method | Returns | Notes |
|--------|---------|-------|
| `moveTo(x, y)` | `Promise<void>` | Move to coordinates in **millimeters**. |
| `penUp()` | `Promise<void>` | Raise pen. |
| `penDown()` | `Promise<void>` | Lower pen. |
| `setPenHeight(n)` | `Promise<void>` | Servo position 0–1 (1 = fully up, 0 = fully down). |
| `setSpeed(mmPerSec)` | `void` | Synchronous — no await. |
| `currentPosition()` | `Promise<{x, y}>` | Current XY in mm. |

## Coordinates & physical setup

All `moveTo` values are in **millimeters**.

**Physical orientation of this plotter:**
- Home (0, 0) = **top-right** corner of the page
- Plotter X increases → pen moves **down** the page (page Y increases)
- Plotter Y increases → pen moves **left** across the page (page X decreases)

Always work in **drawing space** (top-left origin, x→right, y→down) and apply this transform before every `moveTo()`:

```typescript
const PAGE_WIDTH_MM = 210; // adjust to your paper size

function toPlotter(x: number, y: number): { x: number; y: number } {
  return { x: y, y: PAGE_WIDTH_MM - x };
}
```

`moveTo(0, 0)` to return home is always correct without transformation.

Ensure head is at the top-right home position before calling `connect()`.

## Typical sequence

```typescript
await axi.connect();
axi.setSpeed(50);

const start = toPlotter(10, 10);
const end   = toPlotter(80, 10);

await axi.penUp();
await axi.moveTo(start.x, start.y);
await axi.penDown();
await axi.moveTo(end.x, end.y);   // draws 70mm horizontal line
await axi.penUp();
await axi.moveTo(0, 0);           // return to home
await axi.disable();
```

Commands are internally queued — each awaits the previous. Safe to chain with `await`.

## Analog I/O (rarely needed)

| Method | Returns | Notes |
|--------|---------|-------|
| `analogConfigure(channel, enabled)` | `Promise<void>` | Enable channel 0–12 before reading. Avoid channels 3, 5, 6, 7. |
| `analogRead(channel)` | `Promise<number>` | Returns 0–1 (normalized, 1 = 3.3V). |
| `memoryRead(address)` | `Promise<number>` | EBB RAM byte at address 0–4095. Returns 0–255. |
